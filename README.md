# basic Webpage Project

- Styled with CSS (colors, fonts, spacing,layouts)
- Simple JavaScript interaction(a button with alert)

## Technologies
- HTML
- CSS
- JavaScript

## Usage

2. Open index.html in your browser to view the webpage
